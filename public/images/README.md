# Images

## Directory Structure

```
images/
├── headshot.jpg          # Your professional headshot for About page
├── projects/             # Project screenshots/thumbnails
│   ├── project1.jpg
│   ├── project2.jpg
│   ├── project3.jpg
│   ├── project4.jpg
│   └── project5.jpg
└── icons/                # Custom icons if needed
```

## Image Guidelines

### Headshot
- Recommended size: 500x500px or larger (square)
- Format: JPG or PNG
- Professional, well-lit photo

### Project Images
- Recommended size: 1200x800px or similar aspect ratio
- Format: JPG, PNG, or WebP
- Should showcase the project UI or key visuals
- Keep file sizes under 500KB for optimal loading

### Icons
- SVG format preferred for scalability
- PNG with transparent background as alternative
- Consistent style across all icons
